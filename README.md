**ModestMirror**  AR-Based Hijab Store Web Application

**Introduction**

ModestMirror is an innovative Augmented Reality (AR) based hijab store web application designed to enhance the online shopping experience for modest fashion. It allows users to virtually try on different hijab styles in real time using their device camera, helping them make confident and personalized purchase decisions.

The project bridges the gap between physical and online shopping by providing an interactive and immersive virtual try-on experience.

**Purpose**

The main purpose of ModestMirror is to:

Improve online hijab shopping experience using AR technology

Reduce uncertainty in fashion selection

Provide a virtual try-on system for modest wear

Increase customer satisfaction and engagement

Support digital transformation in fashion retail


**Objectives**

Develop a web-based AR hijab try-on system

Allow users to preview hijab styles using live camera

Provide an easy-to-use shopping interface

Manage products efficiently through admin panel

Ensure responsive design for mobile and desktop users

Store and manage user and product data securely
